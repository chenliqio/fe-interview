## 一、题目拆解
1. 递归：在函数中存在调用函数本身的情况就叫做递归。
```
let arr = new Array(5), i = 0

// 获取区间随机数
function getRandom(min = 2, max = 32) {
  return Math.floor(Math.random() * (max - min + 1) + min);
}

(function setArr(num) {
  if (arr.indexOf(num) < 0) {
    arr[i] = num
    i++
  }
  if (i < arr.length) {
    setArr(getRandom())
  }
})(getRandom())

console.log(arr)


```
